Εμφανίζονται μόνο ενεργοί που ΔΕΝ έχουν ήδη user (1-1).
Για non-admin χρήστες, αυτή η ανάθεση πρέπει να ταυτίζεται με την Υπηρεσία του επιλεγμένου Προσωπικού.
Για Administrator, ο user δεν ανήκει σε Υπηρεσία. Παραμένει όμως συνδεδεμένος με κανονικό Personnel.
Σημείωση
  • Κάθε χρήστης συνδέεται 1-1 με Προσωπικό.
  • Admin βλέπει/διαχειρίζεται τα πάντα.
  • Non-admin βλέπει μόνο την υπηρεσία του.
  • Manager/Deputy προκύπτει από ανάθεση στο ServiceUnit, όχι από checkbox.
  • Admin μπορεί να συνδεθεί με “ουδέτερο” Προσωπικό χωρίς Υπηρεσία / Διεύθυνση / Τμήμα.
